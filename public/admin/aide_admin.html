Premiers Pas

Connexion : Utilisez vos identifiants administrateur pour accéder au panneau d'administration.

Connexion Administrateur

Accédez à l'interface d'administration via l'URL principale du système. Utilisez vos identifiants fournis par l'équipe technique.

Tableau de Bord

Le tableau de bord principal affiche un aperçu des statistiques clés : utilisateurs actifs, votes effectués, candidats et élections en cours.

Navigation

Utilisez la barre latérale pour naviguer entre les différentes sections : Utilisateurs, Élections, Candidats, Statistiques et Paramètres.

Tableau de Bord

Le tableau de bord fournit un aperçu en temps réel des activités du système de vote.

Cartes Statistiques

  • Utilisateurs Actifs : Nombre total d'utilisateurs enregistrés et actifs
  • Votes Effectués : Nombre total de votes enregistrés dans le système
  • Candidats : Nombre total de candidats inscrits
  • Élections Actives : Nombre d'élections actuellement en cours

Graphiques

Visualisez l'activité des votes sur les 7 derniers jours avec des graphiques interactifs Chart.js.

Importation d'Étudiants

Choisir le mode d'importation

Sélectionnez entre "Saisie manuelle" ou "Import fichier" selon vos besoins.

Remplir les informations

Pour la saisie manuelle : nom, prénom, matricule, code d'inscription, école et filière.

Importer le fichier CSV

Pour l'import fichier : préparez un fichier CSV avec les colonnes appropriées et sélectionnez-le.

Gestion des Élections

Types d'élections : Salle, École, Université, Générale

Créer une Élection

Accéder à la section Élections

Cliquez sur "Élections" dans la barre latérale.

Remplir le formulaire

Spécifiez : titre, description, type, dates de début et fin, et école concernée.

Ajouter des candidats

Sélectionnez les candidats éligibles pour cette élection.

Activer l'élection

Changez le statut à "ACTIVÉ" pour permettre les votes.

Gérer les Élections Existantes

Action Description Quand l'utiliser
Modifier Changer les détails de l'élection Avant l'activation de l'élection
Activer/Clôturer Contrôler le statut de l'élection Début et fin de la période de vote
Voir les détails Consulter les informations complètes Pour vérifier la configuration
Supprimer Retirer définitivement l'élection Élections test ou annulées
Attention : Une fois une élection clôturée, les votes ne peuvent plus être modifiés.

Gestion des Candidats

Gérez les candidatures des étudiants souhaitant participer aux élections.

Statuts des Candidatures

  • EN_ATTENTE : Candidature soumise, en attente d'approbation
  • APPROUVÉ : Candidature acceptée, candidat éligible
  • REJETÉ : Candidature refusée

Approuver/Rejeter des Candidatures

Accéder à la section Candidats

Cliquez sur "Candidats" dans la barre latérale.

Vérifier les informations

Consultez le profil, programme et photo du candidat.

Changer le statut

Cliquez sur "Changer statut" et sélectionnez APPROUVÉ ou REJETÉ.

Confirmer

Enregistrez les modifications.

Statistiques : La page affiche le nombre de candidatures en attente, approuvées et rejetées.

Gestion des Utilisateurs

Gérez les comptes étudiants et leurs rôles dans le système.

Fonctionnalités Principales

Liste des Utilisateurs

Visualisez tous les utilisateurs avec pagination et recherche.

Filtres

Filtrez par école, filière, année et statut.

Import CSV

Importez des listes d'étudiants via fichier CSV.

Importer des Étudiants

Préparer le fichier CSV

Créez un fichier avec les colonnes : nom, prenom, matricule, email, telephone, ecole, filiere, annee.

Accéder à l'import

Dans la section Utilisateurs, cliquez sur "Importer CSV".

Sélectionner le fichier

Choisissez votre fichier CSV (UTF-8) et validez l'import.

Statistiques et Résultats

Consultez les statistiques détaillées et les résultats des élections.

Types de Graphiques

  • Circulaires : Répartition des votes par candidat
  • Barres : Comparaison des résultats
  • Lignes : Évolution temporelle des votes
  • Tableaux : Données détaillées exportables

Exporter les Données

Utilisez les boutons d'export pour télécharger les résultats au format PDF ou Excel.

Conseil : Les graphiques sont générés avec Chart.js pour une visualisation optimale.

Dépannage

Problèmes Courants

Erreur de Connexion

Vérifiez vos identifiants et réessayez.

Import CSV Échoue

Vérifiez le format du fichier CSV (UTF-8) et assurez-vous que toutes les colonnes requises sont présentes.

Élection Ne S'affiche Pas

Vérifiez que l'élection est activée et que la période de vote est en cours.

Candidature Non Visible

Assurez-vous que la candidature a été approuvée par un administrateur.
