// 1. Élections de l'utilisateur
GET /api/election/my-elections → Doit retourner les élections accessibles à l'utilisateur

// 2. Candidatures de l'utilisateur  
GET /api/candidats/mes-candidatures → Doit retourner les candidatures de l'utilisateur

// 3. Votes de l'utilisateur
GET /api/vote/my-votes → Doit retourner l'historique des votes**

// 4. Validation token vote
POST /api/vote/validate-token → Doit valider un token de vote

// 5. Vérification éligibilité
GET /api/election/{id}/check-eligibility → Doit vérifier l'éligibilité

**Bonus**

// 6. Statistiques utilisateur
GET /api/users/stats → Statistiques personnelles

// 7. Modification candidature  
PUT /api/candidats/{id} → Modifier une candidature**

// 8. Suppression candidature
DELETE /api/candidats/{id} → Supprimer une candidature**
